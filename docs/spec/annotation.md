# 数据标注

数据标注页为顶部主导航中的独立页面，不隶属于“数据处理”子菜单。

当前数据标注页约定：

- 左侧工具栏包含：`图片文件夹`、`标签文件夹`、`上一张`、`下一张`、`画标注框`、`AI预标注`、`更多设置`。
- `标签文件夹` 当前指 Labelme `.json` 标注目录；默认保存与读取 Labelme `.json`。
- `YOLO` 标注目录不再作为左侧独立按钮显示，而是在“更多设置”中配置；开启自动转换后，同步写入 YOLO `.txt` 标注。
- 右侧配置区包含：`任务类别`、`标注类别`、`管理类别`、当前图片标注列表、图片列表。
- `任务类别` 当前为 `detect` / `obb`，决定保存标签时的最终输出格式。
- `标注类别` 为当前新建标注默认使用的类别；右键菜单也支持修改已选中标注的类别。
- 右键菜单顶部必须包含 `编辑` 模式；仅在 `编辑` 模式下允许直接选中、拖动、编辑已有标注。
- 当前支持的绘制模式包括：
  - `矩形框`
  - `圆形`
  - `镜像有向矩形`
  - `有向矩形`
  - `多边形`
  - `直线扩展`（默认关闭，仅在“更多设置”中开启后显示）
- `镜像有向矩形`：先确定一条中心线，再向两侧同时扩展矩形。
- `有向矩形`：先确定一条边线，再向单侧扩展矩形。
- `直线扩展`：以线段为中心线，按固定像素向左右扩展成 OBB；默认扩展像素为 `10`。
- 点击左侧 `画标注框` 按钮时，应先弹出标注类型选择窗口；窗口最上方单独提供 `编辑` 选项，并与下方各绘制类型之间使用分隔线隔开。
- `圆形` 标注采用两次点击确认：第一次确定圆心，第二次确定半径。
- “更多设置”中提供 `开启连续标注` 开关，默认关闭；关闭时每次完成一个标注后自动回到 `编辑` 模式，开启后保持当前标注类型继续绘制。
- “更多设置”中提供 `开启快捷标注` 开关，默认开启；开启时 `矩形框`、`圆形`、`直线扩展` 支持按住鼠标拖动后松开直接完成，关闭时改为通过多次点击确认。
- “更多设置”中提供 `显示标注名称` 开关，默认关闭；关闭时画布只显示标注图形，不显示对应的类别名称。
- `Esc` 规则：
  - 若当前正在绘制标注，先取消当前标注。
  - 若当前没有正在绘制，但仍处于某个绘制模式，再退回 `编辑` 模式。
- 若当前处于 `编辑` 模式且已选中标注，取消当前选中状态。
- 标注绘制过程中只显示图形预览，不显示类别名称；标注完成成型后才显示类别名称。
- 从 `编辑` 模式切换到任意绘制模式时，自动取消当前标注选中状态。
- 标注完成后默认不选中；如需编辑，用户可在编辑模式下点击标注框选中。
- 连续标注模式下，完成标注后下一次点击开始新标注时，应先取消上一标注的选中状态，同时保持连续绘制模式。
- 选中状态下，控制点常驻显示；悬停到对应标注上时才显示浅红色填充。
- 鼠标悬停到可选中标注或控制点上时，光标应切换为小手样式。
- 除 `编辑` 模式外，选择任意绘制模式后，画布中的光标应显示十字样式；多边形悬停到可用于封闭图形的顶点时仍显示小手样式。
- `矩形框` 绘制模式下，鼠标所在位置额外显示贯穿画布的纯黑色实线水平和垂直十字辅助线，并在短十字光标热点周围留出采样空隙，使系统光标仍依据原始图片背景显示对比色；切换其他绘制模式或离开画布后清除辅助线。
- 控制点样式当前区分悬停态；用户可拖动控制点或直接拖动整个已选中标注。
- 控制点尺寸固定为半径 `5 px`（直径 `10 px`），不随图片分辨率变化；控制点命中范围为可视半径的 `1.6` 倍。
- 除多边形外，绘制过程中的预览图形使用红色实线；有向矩形、镜像有向矩形、直线扩展在未完成前的起始预览线也使用红色实线。
- 多边形绘制过程中需显示所有已确认顶点；鼠标悬停到可用于封闭图形的顶点上时，光标切换为小手样式，点击该顶点后立即封闭图形，并短暂显示浅红色填充作为完成反馈。
- 标注列表显示格式为：`序号.类别-绘制类型（最终格式）`，例如 `1.weld-镜像有向矩形（obb）`。
- 图片列表标题显示为 `图片列表`，右侧显示当前索引/总数，例如 `15/460`。
- 图片列表中每个文件名前显示与 `AI预标注 -> 自定义图片列表` 一致的只读 `QCheckBox` 勾选状态；有标注时勾选，无标注时不勾选。
- 首次进入数据标注页时，页面构造阶段不应立即同步执行整套图片扫描；应在页面首次显示后再启动首轮扫描，让导航切页先完成、页面先变为可交互，再尽快显示当前图片与首批图片列表项。
- 若主窗口已在空闲阶段预热过标注页，则切到标注页时应尽量直接显示第一张图片，避免先看到空画布再闪出首图。
- 切换大图片目录时，页面应优先可交互并尽快显示当前图片与首批图片列表项；其余图片列表项和勾选状态允许后台分批补齐，不得因一次性扫描全部图片与标注而出现明显卡顿。图片列表实现应避免为大量不可见行同步创建真实 `QCheckBox` 子控件。
- 若预热阶段已先渲染首批图片列表项，后续分批补齐与后台标注状态扫描仍必须继续执行，不能停留在首批 20 项。
- 图片列表项若使用行内自定义控件显示勾选框和文件名，则底层 `QListWidgetItem` 本身不应再额外绘制同一份文件名文本，避免出现文件名重影或序号错位。
- 图片列表中的勾选框属于只读状态展示，不用于直接点击切换；视觉上应保持正常启用样式，不应呈现禁用发灰效果。
- 管理类别窗口中的类别列表显示格式为：`0 : weld`。
- 顶部中间不再显示图片路径、标签路径等长路径文本。
- 顶部主导航按钮当前采用仅文字样式，不额外显示各页面图标；字号应略大于默认值，保证在 Windows 桌面环境下清晰易读。
- 左侧工具栏标题上方不再显示单独图形符号。
- 左侧标题 `数据标注` 后显示 `ⓘ`，tooltip 说明当前可通过右键菜单快速切换标注类型，默认读写 Labelme，并可在“更多设置”中开启 YOLO 文件保存。
- 主画布右键菜单顶部当前顺序固定为：`编辑`、分隔线、`矩形框`、`有向矩形`、`镜像有向矩形`、`多边形`、`圆形`、`直线拓展`。
- 当右键命中某个已选中标注时，主画布右键菜单应额外显示 `标注类别` 子菜单，并允许直接切换该标注的类别。
- 主画布右键菜单底部功能项当前约定：
  - 若“更多设置”中的 `右键显示保存YOLO标注` 关闭，且 `自动保存 Labelme JSON` 关闭，则显示单个 `保存`，默认保存 Labelme 标注。
  - 若 `右键显示保存YOLO标注` 开启，则按需分别显示 `保存Labelme标注` 与 `保存YOLO标注`；对应自动保存已开启的项目不再显示。
  - `保存` / `保存Labelme标注` 显示快捷键 `Ctrl+S`。
  - `删除` 仅在当前已有选中标注时显示，并显示快捷键 `Del`。
  - `撤销` 仅在当前图片存在未保存修改时显示，并显示快捷键 `Ctrl+Z`。
  - `取消当前绘制` 仅在当前处于绘制中或仍停留在非 `编辑` 模式时显示，并显示快捷键 `Esc`。
- 主画布右键菜单中 `删除` 采用原生 `QAction` 样式，保证与其他原生菜单项对齐且快捷键列显示一致。
- 数据标注页当前内置可用的 `AI预标注` 弹窗，不再是占位按钮。
- AI 预标注弹窗当前默认大小为 `700 x 620`，最小大小为 `650 x 520`。
- AI 预标注弹窗顶部采用左右两张卡片布局：左侧为“模型与参数”，第一行放模型文件，第二行放 `置信度` 与 `IoU`；右侧为“范围与模式”，包含“标注范围”下拉框与“追加 / 替换”处理模式。
- AI 预标注支持：`当前图片`、`当前及以后图片`、`全部未标注图片`、`全部图片`、`自定义图片` 五种范围；支持 `追加` 与 `替换` 两种处理模式；支持模型类别到当前项目类别的映射；推理结果先直接转换为页面内部标注对象并写回 Labelme `.json`，并可按设置同步写 YOLO `.txt`，不是先生成 YOLO 再反转为 Labelme。
- AI 预标注选择 `当前图片` 时，只允许处理当前图片；不得错误扩展到全部图片。
- AI 预标注执行与模型类别读取都应通过隐藏子进程完成；同一次 AI 预标注窗口会话内应尽量复用同一个后台子进程，避免仅因先读取模型类别就重复加载 `torch`，并在关闭对话框、点击返回或中途停止后安全回收；这些操作都不得导致主窗口退出。
- AI 预标注选择 `自定义图片` 后，右侧计数区切换为 `列表` 按钮；非自定义范围显示 `已选择 x 张图片`。点击 `列表` 后打开图片选择窗口，显示当前图片文件夹内全部图片，文件名前带 `QCheckBox` 勾选状态；列表支持按文件名搜索，并在下方提供 `全选`、`反选`、`全不选` 等批量选择按钮，左下角显示 `已选择 x 张图片`，最终仅处理已勾选图片。
- 自定义图片列表支持按住鼠标左键拖动连续多选/取消；拖到列表上下边缘时会自动滚动，且越靠近边缘滚动越快。
- AI 预标注窗口关闭、返回或右上角关闭后，应持久化当前模型、置信度、IoU、标注范围、处理模式和自定义图片选择，下次打开时恢复。
- AI 预标注的“类别映射”表格必须在窗口缩小时保持可滚动和可选择，不能出现下半部分被裁掉、后续类别无法点击的情况；“标注类别”列中的下拉框文字不得被行高裁切。
- 标注列表支持右键菜单：
  - 顶部显示 `标注类型`，点击后展开类别子菜单，作用与主画布右键中的 `标注类别` 相同。
  - 下方显示 `删除标注`，用于删除当前列表项对应的标注。
- 图片列表支持右键菜单：
  - 保存项遵循主画布右键菜单相同规则：显示 `保存` 或按需显示 `保存Labelme标注` / `保存YOLO标注`。
  - 其余操作包含 `删除所有标注`、`删除图片及标注`，两项采用红色危险操作样式。
- 当 `自动保存 Labelme JSON` 关闭且当前图片存在未保存标注修改时，右下角图片列表中文件名后方应显示红色 `（未保存）` 字样。
- 标注页快捷键当前约定：
  - `A`：上一张
  - `D`：下一张
  - `Del`：删除当前选中标注
  - `Ctrl+S`：保存当前 Labelme 标注
  - `Ctrl+Shift+S`：保存当前 YOLO 标注
