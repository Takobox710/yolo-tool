# 系统设置规格

## 系统状态

系统设置页显示 Python、Torch/CUDA、GPU、显存、CPU、内存、磁盘、依赖模块和程序版本。

程序版本来自源码内的 `APP_VERSION`。冻结态缺少发布清单或环境清单，或运行环境与程序要求不一致时，程序启动会拒绝运行并提示安装完整迁移包；这些兼容状态不在系统设置页单独展示。

## Windows 更新包

- `Full`：首次安装或迁移安装，包含程序、运行环境和基础模型。
- `AppUpdate`：只更新程序 EXE、`app_assets` 和发布清单。
- `RuntimeFull`：同时更新程序 EXE、`app_assets` 和完整 `_internal` 环境，不包含模型和用户数据。

更新包不会覆盖项目设置、最近项目状态、模型、图片、标签和训练结果。普通程序更新前必须先安装迁移版完整包；依赖变化时直接安装包含程序层的 `RuntimeFull` 环境升级包。
